Property Inspection Prioritization - PIP TOOL

The aim of the PIP tool is to address the issues of facilitating AHJ's in prioritizing property inspection. Due to limited funding, only about 10% of properties are typically inspected. NFPA is developing a PIP tool – it’s a simple model that requires a minimum amount of input data. In simple terms, the model will rank the inspection priority of a property based on its occupancy and several other risk factors. We would like your assistance in making the analytics/model behind the tool more “intelligent”.

Some point to be noted before filling out the sheet:

  1. Feedback is “RISK based” only (No local political considerations)
  2. There are 50 property field, mark “1” to select only 20 properties (Prioritizing by risk)
  3. The property scenarios are randomly generated samples
  4. Responses will be anonymous (We will be getting feedback from 1000’s of sources) and no personal information will be published
